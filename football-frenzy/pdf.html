Important information about Football Frenzy

Prior to the examination, you are expected to be familiar with the Football Frenzy API. But if you need to, you can explore the API using the built-in web browser.

Requests in Python Runner

In Python Runner, you import and use requests as follows:

from pyodide.http import pyxhr as requests 
 
url = 'https://football-frenzy.s3.eu-north-1.amazonaws.com/api' 
 
r = requests.get(url) 
 
print(r.json())
